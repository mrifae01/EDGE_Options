# EDGE_Options
I built this application for my own use to help me easily trade and manage options contracts while away from my desk. The app connects to my broker via API and runs a bot that automatically places and manages positions based on strategy parameters I configure — including entry conditions, position sizing, and exit rules. A React/Next.js frontend gives me a clean dashboard to monitor open positions and track P&L in real time, while the Python backend handles all broker communication and bot logic.
Built for personal use.
